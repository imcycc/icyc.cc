# 个人博客网站

该项目采用`React+antd+umi+dva`技术栈进行实现。

## 运行效果

## 如何开始

#### clone项目到本地

```git
$ git clone https://github.com/imcycc/icyc.cc
```

#### 安装依赖

```bash
$ npm install
```

#### 启动项目

```bash
$ npm run start
```

## 功能介绍

大体来说分为主网站和管理系统两部分。

### 主网站

+ [x] 文章列表
+ [x] 文章详情

### 管理系统

+ [x] 登录
+ [x] markdown写文章
+ [x] 标签管理（标签列表、添加、修改、删除）
+ [x] 文章管理（文章列表、添加、修改、删除）
